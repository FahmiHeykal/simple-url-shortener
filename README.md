# Simple URL Shortener 

Project ini adalah aplikasi URL Shortener sederhana menggunakan Golang + PostgreSQL + Gin Gonic.

# Fitur
- Generate URL pendek  
- Simpan URL ke database  
- Redirect URL pendek ke URL asli
